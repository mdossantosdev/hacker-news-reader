- Hacker News API

## 🖥️ Getting Started

1. Clone the repository
```sh
$ git clone https://github.com/mdossantosdev/hacker-news-reader.git
```

2. Go into hacker-news-reader
```sh
$ cd hacker-news-reader
```

3. Install dependencies
```sh
$ npm install
```

4. Start the project
```sh
$ npm start
```
